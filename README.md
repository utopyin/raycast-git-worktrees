# Manage Git Worktrees

Helps managing Git Worktrees. 

## Preferences

Configure `Repositories` in Raycast Extension Preferences.
Provide one or more repository paths (newline or comma separated). The command will list all worktrees for each configured repository.

Configure `Default IDE` and `Default Terminal` to choose which applications are used by item actions in the `View Worktrees` command.

## Shortcuts

- `enter`: Open the selected worktree in IDE
- `cmd + enter`: Copy the path of selected worktree to clipboard
- `cmd + t`: Open the selected worktree in Terminal
